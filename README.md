## 문제 및 풀이 공유 방법
1. 문제 공유자는 하루에 SQL문제 1문제를 포함해 2문제 이상을 discussions에 올립니다.
2. 문제는 사이트 주소를 카피하여 올리거나 간단하게 요약하여 올립니다.
3. 팀원들은 문제를 다 풀면 해당 문제에 해당하는 게시글에 `[문제명](풀이링크)` 형태로 코멘트를 달아주세요.
